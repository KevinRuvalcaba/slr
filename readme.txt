Para correr este programa recuerde que se tienen que instalar execjs
pip install PyExecJS
Comandos para ejectuar el programa se encuentran en quick_test_comands.txt
La sintaxis general para correr el programa es:
python main.py action2.csv goto2.csv producciones2.txt *su-entrada*.txt
El archivo de entrada soporta multiples entradas separadas por saltos de lineas.
